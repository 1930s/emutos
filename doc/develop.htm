2. Development Environment

2.01. Code Development

The OEM hardware-specific code development is supported on a Motorola VME-10 system with a graphics display monitor, 256 Kbytes of additional RAM, a termlnal a VME-400 dual serial port. and a Mouse systems serial mouse. Vou have the option of coding In a cross-development environment under CP/M-68K. or coding in GemDos 1.0 native mode. In native mode. you can use prototypes of new hardware added to the VME-IO as expansion boards. The Apple Lisa will support an alternative native mode development environment

2.0.2. Code transport

You can transport GEM DOS to the target system with the "Kermit" communications program.
