EmuTOS - 256 KB versions

These ROMs are suitable for the following hardware:
- STe
- Mega STe
- emulators of the above

Each ROM contains a single language:

etos256cz.img - Czech
etos256de.img - German
etos256es.img - Spanish
etos256fi.img - Finnish
etos256fr.img - French
etos256gr.img - Greek
etos256it.img - Italian
etos256ru.img - Russian
etos256sg.img - Swiss German
etos256us.img - English

These ROM images have been built using:
make all256
